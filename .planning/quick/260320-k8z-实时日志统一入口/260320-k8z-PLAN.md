2. 添加 `NewInstanceStatusHandler` 函数:
   - 调用 `im.GetInstanceNames()` 获取实例名列表
   - 遍历每个实例，通过 `im.GetLogBuffer(name)` 获取实例信息（从中获取 port）
   - 调用 `lifecycle.IsNanobotRunning(port)` 检测运行状态
   - 返回 JSON: `{"instances": [{"name": "...", "port": 1234, "running": true/false}, ...]}`

3. 需要在 InstanceManager 中添加新方法 `GetInstanceConfigs()` 返回实例配置列表（包含 name 和 port），或者修改 `GetLogBuffer` 使其能返回 port 信息。
   推荐方案: 在 `internal/instance/manager.go` 中添加:
   ```go
   func (m *InstanceManager) GetInstanceConfigs() []config.InstanceConfig {
       configs := make([]config.InstanceConfig, 0, len(m.instances))
       for _, inst := range m.instances {
           configs = append(configs, inst.config)
       }
       return configs
   }
   ```

4. 在 `internal/api/server.go` 中注册新路由:
   ```go
   mux.HandleFunc("GET /api/v1/instances/status", web.NewInstanceStatusHandler(im, logger))
   ```

注意: 导入 `internal/lifecycle` 包用于状态检测
