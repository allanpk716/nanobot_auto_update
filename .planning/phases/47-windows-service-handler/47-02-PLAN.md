1. Add RunService wrapper function to lifecycle package.

In `internal/lifecycle/service_windows.go` (already created in Plan 01), add the RunService function:
```go
// RunService runs the application as a Windows service.
// It blocks until the service handler's Execute method returns (D-09).
func RunService(cfg *config.Config, logger *slog.Logger, version string) error {
    handler := NewServiceHandler(cfg, logger, version)
    return svc.Run(cfg.Service.ServiceName, handler)
}
```

In `internal/lifecycle/service.go` (non-Windows stub, already created in Plan 01), add:

```go
// RunService is a no-op stub on non-Windows platforms.

}
```
