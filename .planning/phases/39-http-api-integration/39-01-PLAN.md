Step 1: Add mutex methods to InstanceManager (internal/instance/manager.go)

Add two new methods after the existing IsUpdating() method (line ~307):

```go
// TryLockUpdate attempts to acquire the update lock using atomic CAS.
// Returns true if the lock was acquired, false if already locked.
// Used by SelfUpdateHandler to share the same isUpdating lock with TriggerUpdate (D-02).
func (m *InstanceManager) TryLockUpdate() bool {
	return m.isUpdating.CompareAndSwap(false, true)
}

// UnlockUpdate releases the update lock.
// Must be called when update completes (success or failure) to prevent deadlock.
func (m *InstanceManager) UnlockUpdate() {
	m.isUpdating.Store(false)
}
```

Define two interfaces for duck-typing (same pattern as TriggerUpdater in trigger.go):

```go
// SelfUpdateChecker is the interface for checking and executing self-updates.
// Satisfied by *selfupdate.Updater via duck typing.
type SelfUpdateChecker interface {
	NeedUpdate(currentVersion string) (bool, *selfupdate.ReleaseInfo, error)
	Update(currentVersion string) error
}

// UpdateMutex is the interface for shared update lock operations.
// Satisfied by *instance.InstanceManager via duck typing (D-02).
type UpdateMutex interface {
	TryLockUpdate() bool
	UnlockUpdate()
	IsUpdating() bool
}
```

HandleUpdate method (POST /api/v1/self-update):
- Per D-01: async execution with 202 Accepted
- Order (per RESEARCH Pitfall 2): TryLockUpdate FIRST, then write response, then start goroutine
1. TryLockUpdate() -- if false, return 409 Conflict via writeJSONError (D-06)
2. Store status as `&SelfUpdateStatus{Status: "updating"}`
3. Write 202 response: `{"status": "accepted", "message": "Self-update started"}` (D-04)
4. Launch goroutine:
   - defer UnlockUpdate()
   - defer panic recovery (per RESEARCH Pitfall 1): recover -> store status "failed" with panic info
   - Call updater.Update(h.version)
   - If error: store `&SelfUpdateStatus{Status: "failed", Error: err.Error()}`, log error
   - If success: store `&SelfUpdateStatus{Status: "updated"}`, log info
