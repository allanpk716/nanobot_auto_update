         * Returns `http.FileServer(http.FS(subFS))`
       - Create `WebPageHandler(im, logger)` http.HandlerFunc that:
         * Extracts instance name from `r.PathValue("instance")`
         * Validates instance exists via `im.GetLogBuffer(instanceName)`
         * Serves index.html from embedded FS (use `fs.Sub` + `fs.ReadFile`)
         * Returns 404 if instance not found (reuse InstanceError pattern)

    2. Create internal/web/static/index.html:
       - DOCTYPE html with lang="zh-CN"
       - Head: meta charset, title "Nanobot Logs - {instance}"
       - Link to style.css
       - Body structure:
         * Header with instance selector placeholder (select#instance-select)
         * Connection status indicator (span#connection-status with CSS classes)
         * Scroll toggle button (button#scroll-toggle)
         * Log container (div#logs with role="log" aria-live="polite")
       - Script src="app.js"

    3. Create internal/web/static/style.css:
       - Use spacing scale from UI-SPEC.md (xs=4px, sm=8px, md=16px, lg=24px, xl=32px)
       - Log container: `height: calc(100vh - 120px); overflow-y: auto; padding: 8px`
       - Connection status classes:
         * `.status-connecting { color: #666; }` (gray)
         * `.status-connected { color: #2563eb; }` (blue-600 accent)
         * `.status-disconnected { color: #dc2626; }` (red-600 destructive)
       - Log line styles:
         * `.log-stdout { color: #000000; }` (black, normal)
         * `.log-stderr { color: #dc2626; font-weight: 600; }` (red, bold)
       - Button styling: padding 8px 16px, border 1px solid #ccc, cursor pointer
       - Typography: body 14px/1.5, heading 20px/1.2

    4. Create internal/web/static/app.js:
       - Initialize from URL path: `const instanceName = window.location.pathname.split('/').pop()`
       - Variables: `let autoScroll = true`, `let eventSource = null`, `let reconnectAttempts = 0`
       - `connectSSE(instanceName)` function:
         * Set status to "connecting" (text: "Connecting...")
         * Create `new EventSource('/api/v1/logs/' + instanceName + '/stream')`
         * `onopen`: set status to "connected" (text: "Connected"), reset reconnectAttempts
         * `onerror`: set status to "disconnected" (text: "Disconnected"), increment reconnectAttempts
         * `addEventListener('stdout', ...)`: append log with class "log-stdout"
         * `addEventListener('stderr', ...)`: append log with class "log-stderr"
         * `addEventListener('connected', ...)`: parse instance name from JSON data
       - `appendLog(message, source)` function:
         * Create div element with class "log-" + source
         * Set textContent to message
         * Append to log container
         * If autoScroll is true, scroll to bottom
       - `updateConnectionStatus(status)` function:
         * Update status indicator text and class
       - Scroll event listener on log container:
         * If scrolled within 50px of bottom, set autoScroll = true
         * Otherwise set autoScroll = false
         * Update button text
       - Toggle button click handler:
         * Flip autoScroll
         * If autoScroll becomes true, scroll to bottom
         * Update button text ("Pause Scroll" / "Resume Scroll")
       - Initialize on DOMContentLoaded: connectSSE(instanceName)

    Key patterns from RESEARCH.md:
    - Use `fs.Sub(staticFiles, "static")` to strip prefix (Pitfall 1)
    - Implement onerror handler (Pitfall 2)
    - Use 50px scroll tolerance (Pitfall 3)
    - Use high contrast red for stderr (Pitfall 4)
