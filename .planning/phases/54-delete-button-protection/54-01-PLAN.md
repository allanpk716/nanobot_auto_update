```

**CSS 变更 1 — 修复 hover 规则 (addresses review concern):** 将 `internal/web/static/style.css` L545 的 `.btn-delete-danger:hover` 选择器改为 `.btn-delete-danger:hover:not(:disabled)`，保持属性不变：

    border-color: #dc2626;
    color: #dc2626;
}
```

这遵循了 `.btn-form-danger:hover:not(:disabled)` (L820) 和 `.btn-form-primary:hover:not(:disabled)` (L769) 的现有模式。没有此 guard 时，禁用的删除按钮悬停仍会显示红色边框/文字，与 `cursor: not-allowed` 形成视觉矛盾。

```css
.btn-delete-danger:disabled {
    opacity: 0.6;
    cursor: not-allowed;
}
```

使用 `cursor: not-allowed` 而非 `cursor: wait`，因为这是"不可操作"语义（与 `.btn-form-danger:disabled` L823-826 一致），而非"操作进行中"语义。
