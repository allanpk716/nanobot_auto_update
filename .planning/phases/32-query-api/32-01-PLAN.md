    Add GetPage method to UpdateLogger in internal/updatelog/logger.go. This method implements newest-first pagination by reverse-indexing the internal logs slice (index 0 = oldest, last index = newest per D-03).

    Method signature:
    ```go
    // GetPage returns a paginated slice of logs in newest-first order.
    // offset=0 returns the most recent logs. limit controls page size.
    // Returns the page of logs and the total count of all logs.
    // offset beyond range returns an empty (non-nil) slice.
    func (ul *UpdateLogger) GetPage(limit, offset int) ([]UpdateLog, int)
    ```

    Implementation details (per D-03 newest-first):
    1. Acquire mu.RLock for thread-safe read access
    2. Get total = len(ul.logs)
    3. If offset >= total OR limit <= 0, return empty slice []UpdateLog{} and total
    4. Calculate reverse indices:
       - end = total - offset  (exclusive upper bound, from slice end)
       - start = end - limit   (from end, go back by limit)
       - If start < 0, set start = 0
    5. Create defensive copy: make([]UpdateLog, end-start) then copy from ul.logs[start:end]
    6. REVERSE the result slice so index 0 = newest:
       - For i, j := 0, len(result)-1; i < j; i, j = i+1, j-1 { result[i], result[j] = result[j], result[i] }
    7. Return result, total

    - Verify ordering by checking that result[0].ID is the newest expected log

    Method signature:
    ```go
    // LoadFromFile reads the JSONL file and loads all records into memory.
    // Called at startup after CleanupOldLogs(), before HTTP server starts.
    // If filePath is empty (memory-only mode) or file does not exist, returns nil.
    // Invalid JSON lines are skipped with a warning log.
    func (ul *UpdateLogger) LoadFromFile() error
    ```

    Implementation details:
    1. If ul.filePath == "", return nil (memory-only mode)
    2. If os.Stat returns os.IsNotExist, return nil (file not created yet)
    3. Acquire ul.mu.Lock() (write lock since we append to ul.logs)
    4. Open file with os.Open (read-only)
    5. Use bufio.NewScanner to stream-read line by line (same pattern as CleanupOldLogs)
    6. Skip empty lines
    7. json.Unmarshal each line into UpdateLog, skip invalid lines with ul.logger.Warn
    8. Append valid records to ul.logs
    9. Log count of loaded records via ul.logger.Info
    10. Return scanner.Err() (nil if scan completed successfully)
