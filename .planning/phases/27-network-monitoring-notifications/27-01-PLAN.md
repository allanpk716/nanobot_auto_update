**Step 1: 创建 internal/notification/manager.go**

实现完整的 NotificationManager 结构体和方法。使用 RESEARCH.md 中的参考实现作为基础，确保：

1. **结构体设计：**
```go
type NotificationManager struct {
    monitor  *network.NetworkMonitor
    notifier *notifier.Notifier
    logger   *slog.Logger

    // 内部状态追踪
    previousState *network.ConnectivityState
    mu            sync.RWMutex

    // 冷却时间管理
    cooldownTimer *time.Timer
    pendingChange *stateChange

    // 生命周期控制
    ctx    context.Context
    cancel context.CancelFunc
}

type stateChange struct {
    from bool  // 之前状态: true=连通, false=不连通
    to   bool  // 新状态
    time time.Time
}
```

2. **核心方法：**
- `NewNotificationManager()` - 创建通知管理器
- `Start(checkInterval time.Duration)` - 启动通知管理器（在独立 goroutine 中运行）
- `checkStateChange()` - 检查连通性状态变化
- `confirmAndNotify(change *stateChange)` - 冷却期满后确认状态并发送通知
- `sendNotification(change *stateChange)` - 异步发送通知
- `getErrorType() string` - 获取错误类型（从 ConnectivityState.ErrorMessage 获取）
- `Stop()` - 停止通知管理器

3. **关键实现细节：**

**冷却时间管理 (per D-01):**
```go
// 检测到状态变化时
if nm.cooldownTimer != nil {
    nm.cooldownTimer.Stop()  // 取消之前的待确认变化
}
nm.pendingChange = change
nm.cooldownTimer = time.AfterFunc(1*time.Minute, func() {
    nm.confirmAndNotify(change)
})
```

**冷却期满确认:**
```go
func (nm *NotificationManager) confirmAndNotify(change *stateChange) {
    currentState := nm.monitor.GetState()
    if currentState == nil {
        return
    }
    // 状态已恢复原值，取消通知（网络抖动）
    if currentState.IsConnected != change.to {
        nm.logger.Info("冷却期内状态已恢复,取消通知",
            "change_to", change.to,
            "current", currentState.IsConnected)
        return
    }
    nm.sendNotification(change)
}
```

**通知内容 (per D-02):**
```go
var title, message string
if change.to {
    // 恢复通知
    title = "网络连通性已恢复"
    message = ""
} else {
    // 失败通知
    title = "网络连通性检查失败"
    message = nm.getErrorType()  // 从 ConnectivityState.ErrorMessage 获取
}
```

**Pushover 未配置处理 (per D-03):**
```go
if !nm.notifier.IsEnabled() {
    direction := "从连通变为不连通"
    if change.to {
        direction = "从不连通变为连通"
    }
    nm.logger.Warn("网络连通性状态变化，但 Pushover 通知未配置。请在 config.yaml 中设置 pushover.api_token 和 pushover.user_key",
        "direction", direction)
    return
}
```

**异步发送通知 (per D-04):**
```go
go func() {
    defer func() {
        if r := recover(); r != nil {
            nm.logger.Error("通知发送 goroutine panic",
                "panic", r,
                "stack", string(debug.Stack()))
        }
    }()

    if err := nm.notifier.Notify(title, message); err != nil {
        nm.logger.Error("发送连通性变化通知失败",
            "error", err,
            "title", title)
    }
}()
```

**Stop() 取消冷却 timer:**
```go
func (nm *NotificationManager) Stop() {
    nm.logger.Info("正在停止通知管理器...")
    if nm.cooldownTimer != nil {
        nm.cooldownTimer.Stop()
    }
    nm.cancel()
}
```
