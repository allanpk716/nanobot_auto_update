   - strings

3. Function validateBearerToken(r *http.Request, expectedToken string) error:
   - Extract Authorization header using r.Header.Get("Authorization")
   - If empty, return error "missing Authorization header"
   - Extract token using strings.TrimPrefix(authHeader, "Bearer ")
   - If TrimPrefix returns same string (no "Bearer " prefix), return error "invalid Authorization header format"
   - Use subtle.ConstantTimeCompare([]byte(token), []byte(expectedToken)) for comparison
   - If comparison != 1, return error "invalid Bearer token"
   - Return nil on success

4. Function AuthMiddleware(expectedToken string, logger *slog.Logger) func(http.Handler) http.Handler:
   - Return middleware function that:
     - Calls validateBearerToken
     - On error: call writeJSONError with 401, "unauthorized", error message, log warning
     - On success: call next.ServeHTTP(w, r)

5. Function writeJSONError(w http.ResponseWriter, status int, errorCode, message string):
   - Set Content-Type header to "application/json"
   - Set status code using w.WriteHeader(status)
   - Encode JSON: {"error": errorCode, "message": message}

Key implementation details (from CONTEXT.md locked decisions):
- Use strings.TrimPrefix to extract token from "Bearer <token>"
- Use subtle.ConstantTimeCompare for constant time comparison (prevent timing attacks)
- Error response format: {"error": "unauthorized", "message": "<detail>"}
